
🇧🇷 Este repositório contém uma coleção de scripts em Python focados na resolução de problemas de física, matemática financeira, álgebra, trigonometria e na geração de malhas gráficas (como folhas quadriculadas e monolog).
Descrição dos Arquivos:

    Baskara.py: Script para resolver equações do segundo grau utilizando a Fórmula de Bhaskara. Calcula as raízes reais ou complexas a partir dos coeficientes a, b e c.

    CossenosTriangulos.py / LeiCossenos.py: Aplicação da Lei dos Cossenos para determinar lados ou ângulos desconhecidos em triângulos quaisquer (não retângulos).

    Escala Richter.py: Script que utiliza equações logarítmicas para calcular a magnitude ou a energia liberada por um terremoto na Escala Richter.

    FolhaMonoLog.py: Script gerador de gráficos ou malhas em escala monologarítmica (um eixo linear e um logarítmico), muito útil para engenharia e análise de funções exponenciais.

    FolhaQuadriculaV1.py: Script para gerar imagens de folhas quadriculadas personalizadas, ideal para impressão ou fundo de exercícios geométricos.

    Funcoes.py: Arquivo auxiliar com funções matemáticas genéricas e ferramentas modulares reutilizadas pelos demais scripts.

    JSimplesCompostos.py: Calculadora financeira para simular e comparar a evolução de capitais sob regimes de Juros Simples e Juros Compostos.

    LancamentoVertical.py: Script de simulação de física clássica (cinemática) que calcula o comportamento, altura máxima e tempo de voo de um corpo em lançamento vertical.

Bibliotecas Usadas:

Para que esses scripts funcionem (inclusive no celular pelo Pydroid 3), as principais bibliotecas que precisam estar instaladas são:

    matplotlib (fundamental para os scripts de folhas e malhas gráficas)

    numpy (essencial para cálculos em matrizes e funções matemáticas avançadas)

    math (nativo do Python, usado para raízes e trigonometria básica)

🇺🇸 This repository contains a collection of Python scripts focused on solving physics problems, financial mathematics, algebra, trigonometry, and generating custom graph templates (such as checkered and semi-log sheets).
Files Description:

    Baskara.py: A script to solve quadratic equations using Bhaskara's formula. It calculates real or complex roots based on a, b, and c coefficients.

    CossenosTriangulos.py / LeiCossenos.py: Application of the Law of Cosines to determine unknown sides or angles in oblique (non-right) triangles.

    Escala Richter.py: A script using logarithmic equations to calculate the magnitude or energy released by an earthquake on the Richter Scale.

    FolhaMonoLog.py: A script to generate semi-logarithmic (mono-log) graphs or grids (one linear axis and one log axis), useful for engineering and exponential data analysis.

    FolhaQuadriculaV1.py: A script to generate customized checkered grid images, ideal for printing or as a background for geometry exercises.

    Funcoes.py: A helper file containing generic mathematical functions and modular tools reused across other scripts.

    JSimplesCompostos.py: A financial calculator to simulate and compare capital growth under Simple and Compound Interest systems.

    LancamentoVertical.py: A kinematics physics simulation script that calculates the behavior, maximum height, and flight time of an object in vertical motion.

Dependencies / Libraries:

To run these scripts successfully, make sure you have the following libraries installed:

    matplotlib (required for generating the graph papers and grid templates)

    numpy (required for vector calculations and advanced mathematical functions)

    math (Python's built-in library for standard trigonometric and root operations)
